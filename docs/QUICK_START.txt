🚀 SETUP (Do this once):

   1. Install dependencies:
      → npm install

   2. Get API keys from alpaca.markets

   3. Add keys to .env file:
      ALPACA_API_KEY=your_key_here
      ALPACA_API_SECRET=your_secret_here
      ALPACA_PAPER=true

   4. Test it works:
      → npm test

   5. Run the bot:
      → npm start
